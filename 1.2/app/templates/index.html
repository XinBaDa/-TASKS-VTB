Хранилище "Key-Value"

Получить все данные хранилища WEB

http://{HOSTNAME:PORT}/all

Получить все данные хранилища CURL

curl -i -X GET http://{HOSTNAME:PORT}/all

Получить данные хранилища по ключу WEB

http://{HOSTNAME:PORT}/read?key=test

Получить данные хранилища по ключу CURL

curl -i -GET http://{HOSTNAME:PORT}/read?key=test

Добавить данные в хранилище CURL

curl -i -H "Content-Type: application/json" -X POST -d '{"key": "value"}' http://{HOSTNAME:PORT}/write
